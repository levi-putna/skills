# General Writing

When rewriting or drafting text, produce clean, simple prose. Avoid ornate or overly complicated words unless the context genuinely needs them. If you use an acronym, spell it out on first use unless it is already widely understood (API, URL, HTML, etc.).

## Remove

- Filler transitions: "furthermore," "additionally," "notably," "it's worth noting," "in conclusion," "in summary"
- AI clichés: "delve into," "tapestry," "multifaceted," "unlock the potential," "in today's world"
- Excessive hedging and cautious qualifiers
- Em-dashes used as connectors (replace with rewording)
- Vague adjectives: "significant," "important," "effective" — use specific ones instead

## Add

- Varied sentence rhythm: mix short punchy sentences with longer ones. Break predictable uniformity.
- Concrete details and specifics over generic language
- A natural, slightly informal register appropriate to the context
- Opinions or voice where the original has none
- Occasional imperfect phrasing — natural writing isn't grammatically pristine

## Structure

- Vary paragraph length. Avoid paragraphs of identical size.
- Don't open paragraphs with the same type of sentence repeatedly
- Resist the urge to organise everything into tidy lists

## Formatting rules

- Never use em-dashes (—), ellipses (…), or bullet points mid-paragraph

## Output

Return only the rewritten text, no explanation.
